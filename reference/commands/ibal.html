3.1.15. Ibal: set type of ionisation balance¶

3.1.15.1. Overview¶

For the plasma models, different ionisation balance calculations are possible.

Currently, the default set is Urdampilleta et al. (2017) The Table below lists the possible options.

Ionisation balance modes¶

Abbrevation

Reference

reset

default (=u17)

ar85

Arnaud & Rothenflug (1985)

ar92

Arnaud & Raymond (1992) for Fe,

Arnaud & Rothenflug (1985) for the other elements

bryans09

Bryans et al. (2009)

u17

Urdampilleta et al. (2017) (default)

3.1.15.2. Syntax¶

The following syntax rules apply:

ibal #a : Set the ionisation balance to set #a with #a in the table above.
ibal show : Show the currently active ionisation balance.

3.1.15.3. Examples¶

ibal reset : Take the standard ionisation balance
ibal ar85 : Take the Arnaud & Rothenflug ionisation balance

SPEX Help Center¶

Welcome to the SPEX help center. How can we help you?

I want to download and install SPEX

See How to install SPEX for a list of install options. We distribute pre-compiled binaries, docker images, and the source code.

How do I learn SPEX?

You can start by reading How to run SPEX. If you like to practice, see Exercises.

How do I do … in SPEX?

  • Quick search this website (search box on the left).

  • Check the list of analysis examples in SPEX: Analysis threads.

  • Check our issue-tracker for questions that are similar to yours.

I have another question or problem with SPEX

Check Help & troubleshooting first. If you cannot find it, then please ask your question or report the problem through our issue-tracker on Github. We will try to help you as quickly as possible.
